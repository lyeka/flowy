## 成员清单

circular-progress.jsx: SVG 圆环进度组件，用于显示项目完成进度，支持自定义尺寸/线宽，默认使用系统主题色 var(--primary)，300ms ease-out 过渡动画
context-menu.jsx: shadcn/ui 右键菜单组件，基于 @radix-ui/react-context-menu，用于项目卡片操作（设置/归档/删除）

## shadcn/ui 组件

本项目使用 shadcn/ui 组件库，组件直接添加到此目录，包括：
- button, input, textarea, select, checkbox, radio, switch, slider
- dialog, sheet, dropdown-menu, popover, tooltip
- scroll-area, separator, tabs, avatar, badge
- sonner (toast 通知)
